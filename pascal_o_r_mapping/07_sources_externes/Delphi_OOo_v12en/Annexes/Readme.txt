This zip file contains a toolbox to drive OpenOffice.org application from Delphi.
It is necessary to have OpenOffice.org application installed.

The zip contains :
- document OOo_COM_Delphi_v12en.odt which gives complete explanations.
- units and forms for the Delphi project TestOpen.dpr which is a programming example
- document ConstEnums_en.odt which is used to create unit OOoConstants.pas

Utilisateurs de langue francaise :
